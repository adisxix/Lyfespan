# Lyfespan

Lyfespan is an AI-powered lifespan prediction app that takes your health, lifestyle, and habit data through a clean multi-step form and returns a personalized estimate of how long you'll live — along with a health score, country averages, birthday facts, a famous body twin, and five actionable tips to add more years to your life.

# Features

- Multi-step form — name, DOB, country, body stats, diet, activity, habits, health conditions, and self-assessment
- AI-powered results via Claude API
- Estimated lifespan + years remaining
- Health score out of 5
- Country average life expectancy (male & female)
- Birthday historical fact
- Famous body twin based on your stats
- 5 personalized recommendations to live longer
- Local time display

# Tech Stack

- HTML
- Tailwind CSS (UI components via CDN)
- JavaScript (Vanilla)
- OpenAI API (chatgpt) — AI integration
- Google Fonts — typography
- Tailwind CSS UI Verse Library
